"Sober Driver PRO" 应用隐私政策
1.
简介: 本政策说明了由 NextGen Logic Studio 开发的 "Sober Driver PRO" 如何处理用户数据。
2.
数据收集: "Sober Driver PRO" 不收集、不存储、也不分享任何个人身份信息。
3.
第三方服务: 为了提供广告支持和改进产品，NextGen Logic Studio 使用了 Yandex Mobile Ads 和 AppMetrica SDK。这些服务可能会收集匿名的设备标识符。
4.
数据安全: NextGen Logic Studio 致力于保护用户隐私，除法律要求外，不会将数据泄露给第三方。
